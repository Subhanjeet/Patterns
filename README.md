# patterns  
patterns are logic builders 😤 — print them, learn them, love them. no fluff, just brain gains.

🔥 what is this?  
this repo is a collection of *printing pattern examples* meant to drill foundational logic in programming. if you’re here to get better at loops, control flow, and thinking like a coder, you’re in the right hellscape.

✨ features  
📌 pattern practice  
- tons of java pattern outputs — numbers, shapes, craziness  
- each file is a standalone pattern you can run  
- perfect for beginners to flex logic muscles

📌 learn logic  
- every pattern teaches a twist of loops & conditions  
- printing patterns = canonical brain training  
- helps with interviews & fundamentals, not just syntax

📁 what’s inside  
- `pattern.java` — your entry-level basics  
- `pattern2.java` → `pattern19.java` — progressive pattern examples  
- `.class` files included but you’ll want the `.java` ones

🚀 quick start  
```bash
# clone it
git clone https://github.com/Subhanjeet/Patterns.git

# go into folder
cd Patterns

# compile a pattern
javac pattern.java

# run it
java pattern
